Angular Developer

This project aims to collect the most common / important concepts one should learn to become an Angular developer.


Inspired by ["Web Developer Roadmap"](https://github.com/kamranahmedse/developer-roadmap) and ["React Developer Roadmap"](https://github.com/adam-golab/react-developer-roadmap).

Roadmap#

Topics covered + suggested resources#

  1. The basics:
    • HTML
      • marksheet.io
      • MDN on HTML
    • CSS
      • marksheet.io
      • MDN on CSS
    • JavaScript
      • "You Don't Know JS"
      • NodeSchool.io
      • ES6 Overview in 350 Bullet Points
  2. General Development Skills
    • GIT
      • "Learn Git Branching"
    • HTTP/HTTPS
    • Learn to search for solutions
    • Terminal usage
    • Data Structures and Algorithms
    • Design Patterns
  3. Build tools
    • Package Managers
      • npm
      • Yarn
    • Angular CLI
      • Angular CLI Wiki
    • Bazel
    • Rollup
    • Task runners
      • npm scripts
      • gulp
  4. Styling
    • CSS Preprocessors
      • Sass/SCSS
      • PostCSS
      • Less
    • CSS Frameworks
      • Bootstrap
      • Zurb Foundation
      • Bulma
      • Semantic UI
      • Tailwind CSS
    • Design Systems
      • Angular Material
      • Clarity Design System
      • Ant Design
  5. Architecture
    • Schematics
    • Building Libraries
    • @angular/elements
  6. Coding style
    • Angular Styleguide
    • TSLint
    • Prettier
  7. Fundamental 3rd parties
    • TypeScript
      • "TypeScript Deep Dive"
      • Typescript Docs
    • RxJS
      • The introduction to Reactive Programming
      • RxJS manual
      • RxJS In-Depth (video)
  8. State Management
    • @ngrx
      • /effects
      • /entity
      • ngrx-data
    • NGXS
  9. Progressive Web Apps
    • @angular/pwa
    • Workbox
  10. Utility Libraries
    • Lodash
    • Moment.js
  11. Testing
    • Unit Testing
      • Jasmine
      • Karma
      • Jest
    • E2E Testing
      • Protractor
      • cypress.io
  12. Non-browser Environments
    • Mobile
      • @angular/pwa (ng add @angular/pwa)
      • NativeScript
      • Ionic
    • Angular Universal
    • Electron
